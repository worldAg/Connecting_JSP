* 필요 라이브러리
  * 파일 업로드 **cos.jar**
  * 자바 오브젝트 json 변환 **gson-2.8.5.jar**
  * JSP 표준 태그 **jstl-1.2.jar**
  * 메일 전송 **mail.jar**
  * 오라클 데이터베이스 jdbc 드라이버 **ojdbc6.jar**
